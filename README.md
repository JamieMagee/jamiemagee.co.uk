# jamiemagee.co.uk

Source for my personal site. Built with [Hugo](https://gohugo.io/) using a fork of the [hyde-x](https://github.com/zyro/hyde-x) theme. Deploys to GitHub Pages on push to `main`.

## Running locally

```sh
hugo server -D
```

## New posts

```sh
hugo new blog/my-new-post.md
```

Posts live in `content/blog/`.
